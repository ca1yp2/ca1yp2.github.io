VocaBank 개인정보 처리방침

최종 수정일: 2026-06-12

1. 수집하는 정보

VocaBank는 기능 제공을 위해 다음 정보를 저장할 수 있습니다.

  • Discord 사용자 ID
  • Discord 서버 ID 및 채널 설정 정보
  • 사용자가 등록한 단어, 뜻, 예문
  • 정답 수, 오답 수, 오답 단어 기록 등 학습 통계

2. 정보 사용 목적

수집된 정보는 단어 퀴즈 제공, 점수 표시, 오답 복습, 지정 채널 설정 등 VocaBank 기능 제공을 위해서만 사용됩니다.

3. 정보 공유

VocaBank는 저장된 정보를 제3자에게 판매하거나 제공하지 않습니다.

4. 데이터 보관

데이터는 봇이 운영되는 동안 저장될 수 있으며, 봇 운영 중단 또는 서버 관리자의 요청에 따라 삭제될 수 있습니다.

5. 데이터 삭제 요청

사용자는 서버 관리자 또는 봇 운영자에게 요청하여 자신의 학습 기록 또는 등록 데이터를 삭제할 수 있습니다.

6. 보안

VocaBank는 저장된 데이터를 서비스 운영 목적에 한해 사용하며, 합리적인 수준의 보안 조치를 적용합니다.

7. 문의

개인정보와 관련된 문의는 봇 운영자에게 Discord를 통해 연락해 주세요.
