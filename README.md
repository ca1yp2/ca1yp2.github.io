# ca1yp2 Portfolio (Build Version)

프론트엔드 개발자 **ca1yp2**의 포트폴리오 사이트입니다.  
이 저장소는 **정적 배포용 빌드 파일(`build/`)만 포함**하고 있습니다.

> 원본 소스 코드 저장소가 필요하신 경우 [sdh2683@naver.com](mailto:sdh2683@naver.com) 으로 별도 문의 바랍니다.

---

## 🌐 배포 목적

- 정적 호스팅 (예: GitHub Pages, Netlify, Vercel 등)을 위한 최종 빌드 파일
- `npm run build` 결과물만 포함되어 있으며, 소스 코드(`src/`)는 포함되어 있지 않습니다
- 이 저장소는 **React 기반으로 개발된 프로젝트의 빌드 결과물**입니다  
  (`create-react-app`을 사용해 생성됨)

---

## 🛠 사용 기술 (요약)

- HTML/CSS
- JavaScript (React 기반으로 빌드됨)
- Bootstrap
- AOS (Animate On Scroll)

---

## 🎨 템플릿 출처

이 사이트는 [BootstrapMade](https://bootstrapmade.com/easyfolio-bootstrap-portfolio-template/)의 **EasyFolio** 템플릿을 기반으로 제작되었습니다.  
해당 템플릿은 BootstrapMade의 저작권 하에 있으며, **템플릿 저작권 고지(Footer 주석 등)는 유지되고 있습니다**.
